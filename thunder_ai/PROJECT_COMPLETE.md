## 📋 Setup Required

### 1. Get Gemini API Key
- Visit: https://makersuite.google.com/app/apikey
- Create API key
- Add to `lib/core/constants/app_config.dart`

### 2. Setup Firebase
```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure Firebase
flutterfire configure
```

Or manually:
- Create Firebase project
- Add Android app (package: `com.example.thunder_ai`)
- Download `google-services.json` → `android/app/`
- Enable Firestore in test mode

### 3. Run the App
```bash
flutter pub get
flutter run
```

## 📱 How It Works

1. **Create Chat**: Tap FAB → New chat created in Firestore
2. **Send Message**: Type & send → Saves to Firestore
3. **AI Response**: Gemini generates response → Saves to Firestore
4. **Auto Title**: First message generates chat title
5. **Real-time**: All changes sync instantly
6. **Time Groups**: Chats auto-group by Today, Yesterday, etc.

## 📊 Database Structure

```
Firestore
└── chats/
    └── {chatId}/
        ├── userId: "demo_user"
        ├── title: "AI-generated title"
        ├── lastMessage: "Last message text"
        ├── updatedAt: timestamp
        └── messages/
            └── {messageId}/
                ├── sender: "user" | "ai"
                ├── text: "Message content"
                ├── timestamp: timestamp
                └── isMarkdown: true | false
```

## 🐛 Troubleshooting

### "Gemini API key not configured"
→ Update `app_config.dart` with your API key

### "Firebase initialization error"
→ Run `flutterfire configure`

### "Permission denied"
→ Check Firestore security rules (should be in test mode)

### No AI responses
→ Check internet connection and API key
