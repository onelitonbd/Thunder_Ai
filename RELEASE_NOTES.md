# Thunder AI v1.0.0 Release Notes

## 🚀 Release Date
April 3, 2025

## 📦 What's New

### Features
- AI-powered chat interface with Google Gemini
- Real-time messaging with Firebase Firestore
- Modern, clean UI design
- Chat history management
- Time-based chat grouping (Today, Yesterday, Last 7 Days, etc.)
- Dark and Light theme support

## 📥 Installation

Download `thunder-ai-v1.0.0.apk` and install on your Android device.

## 🔧 Requirements
- Android 5.0 (API 23) or higher
- Internet connection for AI features

## 🔐 Setup Required
1. Get Gemini API key from https://makersuite.google.com/app/apikey
2. Add to `lib/core/constants/app_config.dart`
