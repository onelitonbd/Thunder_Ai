## 📱 Installation

### For Users
1. Go to: https://github.com/onelitonbd/Thunder_Ai/releases/tag/v1.0.0
2. Download `thunder-ai-v1.0.0.apk`
3. Install on Android device (API 23+)
4. Configure Gemini API key and Firebase

### For Developers
```bash
git clone https://github.com/onelitonbd/Thunder_Ai.git
cd Thunder_Ai/thunder_ai
flutter pub get
flutter run
```

---

## 🔧 Setup Required

Users need to configure:
1. **Gemini API Key** - Get from https://makersuite.google.com/app/apikey
2. **Firebase Project** - Run `flutterfire configure`

See `QUICK_START.md` for detailed instructions.
